# מערכת חישוב שכר ישראלית (Payroll System)

מערכת greenfield לחישוב שכר עבור עובדים בישראל, כולל:

## תכונות עיקריות

- **חישוב שכר בסיסי**: שעות × תעריף
- **שעות נוספות**: 125% / 150% / 175%
- **שבת וחג**: תעריפים מיוחדים
- **בונוסי אירועים**: בונוסים חד-פעמיים
- **תוספות**: נסיעות, אוכל, ביגוד וכד'
- **הפרשות**:
  - פנסיה 6.5% (מעסיק) + 6% (עובד)
  - פיצויים 8.33%
  - קרן השתלמות 7.5% / 2.5%
  - ביטוח לאומי
  - מס הכנסה (מדרגות 2026)
- **חופשה, מחלה, מילואים**: צבירה וניצול
- **דוחות**:
  - תלוש שכר חודשי PDF (RTL בעברית)
  - טופס 106 שנתי
  - דוח 102 חודשי
  - דוח 126 חודשי

## מבנה

```
src/
├── schemas/      - מודלי נתונים
├── engine/       - מנוע חישוב
├── reports/      - דוחות PDF
├── api/          - REST API
├── ui/           - ממשק משתמש RTL
└── utils/        - כלי עזר
```

## הרצה

```bash
npm install
npm start          # מפעיל שרת על פורט 3000
npm run demo       # הרצת דמו
npm test           # בדיקות
```
