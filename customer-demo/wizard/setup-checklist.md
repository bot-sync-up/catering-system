# Setup Checklist — Onboarding

> רשימה לאישור מנהל ה-onboarding לפני Go-Live של לקוח חדש.

## חברה ופרטים

- [ ] שם החברה, ח.פ., כתובת — וודאו במרשם החברות
- [ ] לוגו הועלה (PNG/SVG, רזולוציה גבוהה)
- [ ] תעודת רישום עסק (PDF) עלתה
- [ ] טלפון העסק מאומת (קוד SMS)
- [ ] חשבון בנק מאומת (העברת 1 אגורה)

## משתמשים

- [ ] Owner הוגדר, פרטי קשר מאומתים
- [ ] לפחות 1 Admin נוסף קיים (כיסוי)
- [ ] כל המשתמשים הקריטיים הופעלו 2FA
- [ ] CSV מהיר של עד 50 משתמשים עלה בהצלחה
- [ ] שיוך תפקידים בוצע — לבדוק עם הלקוח

## אינטגרציות

- [ ] WhatsApp Business מחובר ופועל (שליחת הודעת מבחן)
- [ ] PBX מחובר (אם רלוונטי)
- [ ] חשבונית ירוקה / חשבשבת מחובר
- [ ] Pelecard / Cardcom מחובר (תשלום מבחן 1₪ + החזר)
- [ ] Google Calendar / Outlook מסונכרן

## נתונים

- [ ] קטלוג עלה (לפחות 20 פריטים)
- [ ] לקוחות עלו (לפחות 10 או ייבוא CSV מלא)
- [ ] ספקים עלו
- [ ] חוזים פעילים עלו (אם רלוונטי)
- [ ] תמונות עלו ועברו אופטימיזציה

## הדרכה

- [ ] 8 שעות הדרכה הושלמו
- [ ] צוות עבר Quick Start
- [ ] צוות הפיננסי עבר הדרכת חשבוניות + מע"מ
- [ ] עובדי שטח התקינו את האפליקציה
- [ ] מנכ"ל ראה דשבורד BI

## טכני

- [ ] גיבוי ראשון רץ בהצלחה
- [ ] webhook tests עברו
- [ ] SSL תקין
- [ ] DNS תקין (אם תת-דומיין מותאם)
- [ ] performance test (זמן טעינה <2s)

## חוזה ותשלום

- [ ] הסכם נחתם דיגיטלית
- [ ] חשבונית מס הופקה
- [ ] תשלום ראשון התקבל / הוראת קבע פעילה
- [ ] כתובת לחיוב אומתה

## CSM

- [ ] CSM הוקצה
- [ ] תזמון 7 days check-in
- [ ] תזמון 30 days check-in
- [ ] תזמון 90 days QBR
