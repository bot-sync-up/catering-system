# שלב 5 — לקוח ראשון

**זמן משוער:** 4 דקות | **חובה:** ✘

## רישום ידני

- שם לקוח (אישי או חברה)
- ח.פ. / ת.ז.
- אנשי קשר (מספר, מייל, טלפון)
- כתובת/כתובות (חיוב, אספקה)
- תנאי תשלום ברירת מחדל
- הערות פנימיות

## ייבוא מ-CRM קיים

- HubSpot — OAuth
- Salesforce — API key
- Monday — API key
- אקסל — העלאת CSV

## נורמליזציה אוטומטית

ה-AI מנקה:
- כתובות (פורמט אחיד)
- מספרי טלפון (פורמט בינלאומי)
- שמות (אותיות גדולות בתחילת מילה)
- מציין כפילויות

## תיוגים

- VIP, חוזר, חדש
- ענף לקוח (משפחות, ארגונים, חינוך)
- מקור (המלצה, אתר, פייסבוק)
