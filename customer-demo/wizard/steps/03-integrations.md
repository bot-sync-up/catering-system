# שלב 3 — אינטגרציות

**זמן משוער:** 8 דקות | **חובה:** ✘ (אך מומלץ מאוד)

## אינטגרציות מומלצות

### תקשורת לקוחות
- **WhatsApp Business** — חיבור ל-Meta Business, בחירת מספר עסקי
- **PBX (IP Sales)** — שתי לחיצות, אישור מקובץ של ה-PBX
- **SMS Gateway** — חיבור לספקי SMS ישראליים (אינפורגייט, סלסקום SMS)
- **דוא"ל יוצא** — SendGrid/Mailgun עם אימות SPF+DKIM אוטומטי

### פיננסים
- **חשבונית ירוקה** — API token + secret
- **חשבשבת** — ייצוא אוטומטי או ייבוא דו-כיווני
- **Pelecard / Cardcom** — קבלת תשלומים
- **בנק (PSD2 ישראלי)** — חיבור באמצעות Open Banking לקבלת תנועות

### תפעול
- **Google Calendar** — סנכרון אירועים
- **Outlook 365** — סנכרון יומן + מיילים
- **Google Maps** — ניווט לנהגים
- **Dropbox / Google Drive** — אחסון מסמכים

### רגולציה
- **מס הכנסה** — חיבור Sync-Up Direct לדיווח 102/126/856
- **ביטוח לאומי** — דיווח 102
- **מע"מ** — דיווח דו-חודשי אוטומטי

## תהליך החיבור (לכל אינטגרציה)

1. לחיצה על "חבר" — פתיחת מודל OAuth/API key
2. אישור הרשאות
3. בדיקת חיבור (Health check)
4. אישור פעולה

## הסבר ללקוח

> אינטגרציה אחת = 4-8 שעות עבודה חיסכון חודשי. אם אינטגרציה לא ברשימה — נכין אותה תוך 5 ימי עסקים בתעריף פיתוח קבוע.
