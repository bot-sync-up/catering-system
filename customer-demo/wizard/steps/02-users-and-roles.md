# שלב 2 — משתמשים ותפקידים

**זמן משוער:** 5 דקות | **חובה:** כן (מינימום משתמש אחד)

## תפקידי ברירת מחדל

| תפקיד | הרשאות |
|---|---|
| Owner | מלא, ללא הגבלה |
| Admin | מלא חוץ מ-billing |
| Manager | ניהול אירועים + צוות, ראייה ב-BI |
| Accountant | פיננסים מלא, ראייה בלבד בשאר |
| Agent | מכירות + לידים, ראייה בתפעול |
| Worker | אפליקציית עובד בלבד |
| Driver | אפליקציית נהג בלבד |
| Customer | פורטל לקוחות בלבד |

## פעולות במסך

- הוספת משתמש חדש (שם, מייל, טלפון, תפקיד)
- הזמנה במייל + SMS עם Magic Link
- הגדרת 2FA חובה לתפקידים מסוימים (Owner, Admin, Accountant)
- מצב massive — העלאת CSV של עד 500 משתמשים בבת אחת

## CSV Template

```csv
שם_מלא,מייל,טלפון,תפקיד,מספר_עובד,יחידה
דוד כהן,david@company.co.il,0501234567,Manager,001,מטה
שרה לוי,sara@company.co.il,0521234567,Agent,002,מכירות
```

## אינטגרציה ל-HR קיים

אם הלקוח מנהל עובדים במערכת קיימת (חילן, מיכפל, מלם), ניתן ליבא רשימה אוטומטית.

## הסבר חשוב ללקוח

> כל משתמש מקבל בדיוק את ההרשאות שצריך. עובד שטח לא יכול לראות שכר של אחרים. ה-Audit Log שומר כל פעולה לתקופה של 7 שנים (לפי חוק).
