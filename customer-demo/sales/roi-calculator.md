## גיליון 2 — חישוב (נסתר מהלקוח)

### חיסכון מאדמיניסטרציה
```
שעות חודשיות = שעות שבועיות × 4.33
= 32 × 4.33 = 138.6 שעות
ערך חודשי = 138.6 × 85 = ₪11,781
ערך שנתי = ₪141,372

חיסכון של 70% עם Sync Up = ₪98,960
```

### צמצום שגיאות חשבוניות
```
חיובים שנתיים מוערכים = מחזור / 5,000 = 1,000 חיובים
שגיאות = 1,000 × 4% = 40
עלות ממוצעת לשגיאה = ₪1,200 (זמן+החזר+מותג)
עלות שנתית = ₪48,000

צמצום של 95% עם Sync Up = ₪45,600
```

### חיסכון מחיוב כפול לספקים
```
חשבוניות ספק שנתיות = 600
כפילויות = 600 × 12% = 72
סכום ממוצע = ₪2,800
סך כפילויות = ₪201,600

מתוכן ~30% מתגלות בזמן ומחזירות, מסת חיסכון אמיתית = ₪141,120
צמצום של 98% עם OCR ו-deduplication = ₪138,298
```

### שיפור DSO (Days Sales Outstanding)
```
שיפור = 67-32 = 35 ימים
מחזור יומי = 5,000,000 / 365 = ₪13,699
שחרור הון חוזר = 35 × 13,699 = ₪479,452
ערך כלכלי (8% עלות הון) = ₪38,356 בשנה
```

### חיסכון בחיפוש מידע (אדמיניסטרציה)
```
שעות חודשיות = 8 × 4.33 = 34.6 שעות
ערך חודשי = 34.6 × 85 = ₪2,941
ערך שנתי = ₪35,292

חיסכון של 85% עם search+AI = ₪30,000
```

### תוספת רווחיות מ-BI
```
מחזור × תוספת = 5,000,000 × 3% = ₪150,000 רווח נוסף שנתי
```

### חישוב ניהול אירועים יעיל יותר
```
240 אירועים, כל אחד חוסך 1.5 שעות תיאום
= 360 שעות שנתיות × ₪85 = ₪30,600
```
