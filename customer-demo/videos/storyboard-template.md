## נקודות מפתח לבדיקה

- [ ] Hook ברור ב-3 שניות ראשונות
- [ ] Value proposition מובן תוך 15 שניות
- [ ] לפחות נקודת WOW אחת
- [ ] CTA חזק וברור בסוף
- [ ] subtitles עברית מסונכרנים
- [ ] subtitles אנגלית מסונכרנים
- [ ] לוגו מופיע בתחילה ובסוף
- [ ] URL ברור בסוף, ניתן לקריאה ב-2 שניות
- [ ] אורך מדויק לפלטפורמה (ימשך)

## משאבי הפקה

- **Voice over** — סטודיו מוסכם, רשימה ב-`videos/vendors.md`
- **גופנים** — Heebo Regular/Bold, Assistant (RTL מתאים)
- **צבעים** —

- **לוגו pack** — `assets/brand/logo-pack/`
- **מוזיקה רוילטי-פרי** — Artlist, Epidemic Sound
