# 04 — מדריך משתמש: מנהל (CRM + Orders + Reports)

## 1. כניסה ראשונה למערכת

1. פתח דפדפן (Chrome / Edge / Firefox מעודכנים)
2. גלוש לכתובת: `https://app.yourdomain.com`
3. הזן אימייל וסיסמה שקיבלת
4. במידה והוגדר MFA — אשר את הקוד שמגיע ב-SMS / Authenticator

## 2. לוח הבקרה הראשי (Dashboard)

הלוח מציג בזמן אמת:
- **הזמנות היום** — מספר, סטטוסים, סכום
- **לקוחות פעילים השבוע** — Active Customers
- **הכנסה — חודש שוטף** — מצטבר עם השוואה לחודש קודם
- **התראות** — מלאי נמוך, תשלומים פתוחים, תקלות

## 3. CRM — ניהול לקוחות

### 3.1 הוספת לקוח חדש
1. **Customers ← + לקוח חדש**
2. מלא: שם, ח.פ./ת.ז, טלפון, אימייל, כתובת
3. בחר **סוג לקוח**: פרטי / עסקי / מוסדי
4. **תנאי תשלום**: שוטף+0/30/60, כרטיס אשראי שמור
5. **תווית מחיר** (Price List) — אם רלוונטי
6. שמור

### 3.2 ניהול לקוח קיים
- כרטיס לקוח כולל: פרטים, היסטוריית הזמנות, יתרה, מסמכים, הערות
- **Timeline** — כל אינטראקציה: שיחה, אימייל, הזמנה, תלונה
- **קבצים מצורפים** — חוזים, הסכמים, אישורי מסים
- **תיוג (Tags)** — VIP, חוב, אסטרטגי וכו'

### 3.3 פילוחים ויצוא
- שדה חיפוש חכם (שם / טלפון / ח.פ.)
- מסננים: תווית, סטטוס, יתרה, תאריך הזמנה אחרונה
- **Export** ל-CSV / Excel

## 4. ניהול הזמנות

### 4.1 יצירת הזמנה
1. **Orders ← + הזמנה חדשה**
2. בחר לקוח (חיפוש חכם)
3. תאריך + שעה ליעד
4. הוסף מנות מהקטלוג (חיפוש או סריקה)
5. הערות מיוחדות (אלרגיה, ללא בצל וכו')
6. בחר אופן משלוח: איסוף / מסירה / יעד מיוחד
7. **שמור** — ההזמנה נכנסת לסטטוס "Pending"

### 4.2 סטטוסי הזמנה
- `Pending` — ממתינה לאישור
- `Confirmed` — אושרה, הופקה למטבח
- `Preparing` — בהכנה
- `Ready` — מוכנה לאיסוף/משלוח
- `Out for Delivery` — בדרך
- `Delivered` — נמסרה
- `Cancelled` — בוטלה
- `Refunded` — הוחזר תשלום

### 4.3 פעולות מהירות
- **שכפול הזמנה** — לפי הזמנה קודמת של אותו לקוח
- **הזמנה חוזרת** (Recurring) — שבועית/חודשית
- **פיצול הזמנה** — מספר יעדים
- **מיזוג הזמנות** — אותו לקוח באותו יום

## 5. דוחות (Reports)

### 5.1 דוח מכירות
**Reports ← Sales Report**
- חיתוכים: לפי לקוח / מנה / קטגוריה / משלוח
- טווח תאריכים מותאם
- השוואה לתקופה מקבילה
- ייצוא ל-Excel / PDF

### 5.2 דוח רווחיות
- Revenue − Food Cost − Labor = **Gross Profit**
- אחוזי רווחיות לפי מנה
- זיהוי מנות מפסידות

### 5.3 דוח לקוחות
- Top 20 לקוחות
- Churn Analysis — לקוחות שעזבו ב-90 הימים האחרונים
- AOV per customer

### 5.4 דוח הפצה
- OTD (On-Time Delivery)
- מספר עצירות לנהג
- ק"מ ממוצע למשלוח

### 5.5 דוח כספים
- חייבים פתוחים (Aged Receivables)
- חשבוניות סגורות/פתוחות
- תזרים מזומנים צפוי

## 6. טיפים למנהל יעיל

- בנה **Dashboard מותאם** בעמוד הבית עם 6 ה-KPIs שחשובים לך
- הגדר **התראות מותאמות** (Slack / Email) על חריגות
- בצע **Weekly Review** של דוח רווחיות
- **חודשי**: סקור Churn, NPS, וביצוע ספקים

## 7. קיצורי מקלדת מרכזיים

| קיצור | פעולה |
|---|---|
| `G + O` | מעבר ל-Orders |
| `G + C` | מעבר ל-Customers |
| `G + R` | מעבר ל-Reports |
| `N` | הזמנה חדשה |
| `/` | חיפוש גלובלי |
| `?` | הצגת כל הקיצורים |
