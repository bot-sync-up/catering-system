# 07 — מדריך משתמש: פורטל לקוח

## 1. גישה לפורטל

- **כתובת**: `https://order.yourdomain.com` (subdomain מותאם ללקוח)
- נגיש מ-Desktop, Tablet ו-Mobile (Responsive)
- אין צורך בהורדת אפליקציה — PWA תומך התקנה למסך הבית

## 2. הרשמה / התחברות

### 2.1 לקוח חדש
1. כנס לפורטל ← **"הרשמה"**
2. הזן: שם, אימייל, טלפון, סיסמה
3. אישור OTP ב-SMS
4. השלם פרטי חיוב (אופציונלי בהתחלה)

### 2.2 לקוח קיים
1. הזן אימייל/טלפון + סיסמה
2. או **התחברות מהירה** עם OTP בלבד
3. או Google / Apple Sign-in (אם הופעל)

### 2.3 שכחת סיסמה
- לחץ "שכחתי סיסמה" ← הזן אימייל ← תקבל קישור איפוס

## 3. דף הבית

מציג:
- **קטגוריות תפריט** עם תמונות
- **המנות שלי** — מנות שהוזמנו בעבר (Quick Reorder)
- **מבצעים פעילים**
- **קופונים זמינים** — של הלקוח

## 4. הזמנה חדשה

### 4.1 גלישה בתפריט
1. בחר קטגוריה (לדוגמה: "ארוחות בוקר", "ירקות חיים")
2. לחץ על מנה לפתיחת כרטיס מורחב
3. צפה ב: תיאור, מרכיבים, אלרגנים, ערכים תזונתיים, תמונה
4. בחר כמות + תוספות/הסרות
5. **"הוסף לעגלה"**

### 4.2 חיפוש
- שדה חיפוש כללי בראש העמוד
- מסננים: צמחוני / טבעוני / ללא גלוטן / כשר מהדרין

### 4.3 עגלה (Cart)
- צפייה בכל המנות שנבחרו
- שינוי כמות / הסרה
- הזנת קופון
- בחירת תאריך + שעת יעד
- בחירת שיטת אספקה: **מסירה** / **איסוף עצמי**

### 4.4 Checkout — סיום הזמנה
1. וידוא פרטי משלוח (כתובת + קומה + הערות לנהג)
2. בחירת שיטת תשלום:
   - כרטיס אשראי שמור
   - כרטיס אשראי חדש
   - PayPal / Bit / Apple Pay / Google Pay
   - הוראת קבע (לעסקיים)
3. סיכום סופי ← **"שלם והזמן"**
4. קבלה דיגיטלית נשלחת ב-Email + WhatsApp

## 5. מעקב הזמנה Real-time

לאחר הזמנה:
- מסך מעקב עם 5 שלבים:
  1. **התקבלה** ✓
  2. **בהכנה במטבח**
  3. **מוכנה**
  4. **בדרך אליך** + מפה Real-time עם מיקום הנהג
  5. **נמסרה**
- התראות Push + WhatsApp בכל שלב
- ETA מתעדכן באופן רציף

## 6. אזור אישי (My Account)

### 6.1 ההזמנות שלי
- היסטוריית הזמנות + סטטוס
- כפתור "הזמן שוב" על כל הזמנה קודמת
- הורדת חשבוניות בPDF

### 6.2 הכתובות שלי
- ניהול כתובות שמורות (בית, עבודה, וכו')
- ברירת מחדל

### 6.3 כרטיסי אשראי
- ניהול אמצעי תשלום שמורים (PCI Compliant Tokenization)
- אפשרות הסרה

### 6.4 העדפות
- אלרגיות + העדפות תזונה (קבועות לכל הזמנה)
- הגדרות התראה (אימייל / SMS / WhatsApp / Push)
- שפה (עברית / אנגלית / ערבית / רוסית)

### 6.5 קופונים שלי
- רשימת קופונים זמינים + תאריך תפוגה

## 7. הזמנה חוזרת (Subscription)

### 7.1 יצירה
- בכל הזמנה ← **"הפוך להזמנה קבועה"**
- בחר תדירות: יומית / שבועית (יום) / חודשית
- שעת אספקה קבועה

### 7.2 ניהול
- **My Account ← Subscriptions**
- אפשרות: השהיה זמנית (Pause), שינוי, ביטול
- "Skip Next Delivery" — דילוג על השבוע הבא

## 8. דירוגים וביקורות

- לאחר מסירה — שאלון קצר: ⭐ 1-5
- אפשרות להוסיף תגובה חופשית
- ביקורות חיוביות עוזרות לעסק

## 9. צ'אט תמיכה

- כפתור צ'אט בפינה ימנית תחתונה
- AI Chatbot (24/7) לשאלות בסיסיות
- העברה לנציג אנושי בשעות פעילות

## 10. נגישות

- תאימות מלאה ל-WCAG 2.1 AA
- ניגוד צבעים גבוה (Dark/Light Mode)
- ניווט מקלדת מלא
- תאימות ל-screen readers (NVDA, VoiceOver)

## 11. שאלות נפוצות (FAQ קצר)

**ש: מה זמן האספקה המינימלי?**
ת: בדרך כלל 2 שעות ממועד ההזמנה (תלוי בעסק).

**ש: האם אפשר לבטל?**
ת: כן, עד 30 דקות לפני המועד. לאחר מכן — לפי מדיניות העסק.

**ש: איך אקבל חשבונית?**
ת: אוטומטית באימייל מיד לאחר התשלום + זמינה לתמיד באזור אישי.

**ש: שכחתי את הסיסמה — מה לעשות?**
ת: לחץ "שכחתי סיסמה" בעמוד ההתחברות.
