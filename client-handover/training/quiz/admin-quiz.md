## חלק ב' — שאלות תרחישיות (4 שאלות, נקודה לכל אחת)

**13. לקוח רוצה למחוק את כל הנתונים שלו (GDPR). מה תעשה?**
תשובה: Admin → Data Privacy → Right to be Forgotten → אמת זהות → אישור מחיקה → המערכת מבצעת soft delete + anonymization, ומשאירה רישומים חוקיים (חשבוניות) כנדרש בחוק.

**14. גילית 50 ניסיונות התחברות כושלים מ-IP יחיד ב-5 דקות. מה הצעדים?**
תשובה: בדוק Audit Log → אמת שזה ניסיון Brute Force → הוסף IP ל-Blocklist → הפעל Force Reset Password למשתמש המותקף → אם רחב יותר — הסלמה ל-Security Team.

**15. ה-iCount הפסיק לעבוד פתאום. מה תבדוק?**
תשובה: Integrations → iCount → Health Check → אם נכשל, בדוק שלא פג תוקף ה-API Token → צור Token חדש → עדכן במערכת → Retry סינכרון.

**16. לקוח מבקש שדרוג מ-Growth ל-Pro באמצע החודש. מה תעשה?**
תשובה: Admin → Plan & Billing → Upgrade → הצג Pro-rata charge ללקוח → בקש אישור → ביצוע השדרוג מיידי + חיוב יחסי.
