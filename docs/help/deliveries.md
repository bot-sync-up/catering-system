# רכבים ומשלוחים

## הקצאת רכב

המערכת בוחרת אוטומטית את הרכב הקטן ביותר שיכול לשאת את ההזמנה (תלוי
במשקל ונפח). ניתן לדרוס ידנית.

## מסלול

אם מוגדר Google Maps API, נקודות העצירה מסודרות לפי TSP פשוט (מסלול
הקצר ביותר).

## משלוחי רחפן

לאזורים נבחרים (חיפה, צפון תל אביב) — ניתן להפעיל ספק רחפן חיצוני
(`@syncup/innovation/drone`). כיום stub בלבד בתצוגה.
