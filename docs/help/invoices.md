# חשבוניות וגבייה

## הפקת חשבונית

אוטומטית בעת `delivered`, או ידנית מתפריט ההזמנה.

החשבונית נוצרת ב-iCount (אם הוגדר token) ונשלחת ללקוח במייל + WhatsApp.

## פיוס בנקאי

הזרימו דוח תנועות בנק (CSV) דרך Settings → Bank Reconciliation. המערכת תתאים
תנועות לחשבוניות באוטומציה (exact + fuzzy), ותסמן את שלא הותאם לבדיקה ידנית.

## מע"מ

מאוגוסט 2025 — שיעור מע"מ בישראל 18%. הגדרת ברירת המחדל מתעדכנת אוטומטית
בכל שינוי חוקי על-פי `packages/vat-engine`.
