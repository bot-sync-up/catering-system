# תפריטים ופריטים

## ייבוא מהיר

`Menus → Import CSV` עם הכותרות: `name_he, price_ils, category, allergens`.

## תמונות AI

לכל פריט ניתן ללחוץ "צור תמונה ב-AI" — המערכת תייצר תמונה ברמת סטודיו
בעברית עם תיאור אוטומטי. ניתן לבחור סגנון:
- studio (ברירת מחדל)
- rustic
- minimalist
- luxury
- street-food
- kosher-shabbat

## תצוגת AR

עבור פריטים שיש להם קובץ `.glb`, לקוחות יכולים לצפות תלת-ממדית במציאות
רבודה ישירות מהדפדפן. ראו [innovation](./innovation.md).

## ניהול אלרגנים

המערכת תומכת בכל סוגי האלרגנים הסטנדרטיים + "כשר" + "פרווה" / "חלבי" / "בשרי".
