# התחלת עבודה ב-Sync Up Catering

ברוכים הבאים!

## שלב 1 — הקמת החשבון

1. אישור מייל הזמנה מהמנהל.
2. בחירת סיסמה (או רישום Passkey — מומלץ).
3. השלמת פרטי חברה: שם, ח.פ., כתובת, פרטי בנק.

## שלב 2 — הגדרות עיקריות

- **תפריט בסיסי**: ייבוא מ-CSV או בנייה ידנית.
- **לקוחות**: ייבוא או הוספה ידנית.
- **חשבונית ל-iCount**: חיבור Token (Settings → Integrations).

## שלב 3 — הזמנה ראשונה

1. הזמנות → "חדש".
2. בחרו לקוח / צרו חדש.
3. הוסיפו פריטים מהתפריט.
4. שמרו ושלחו אישור ללקוח (SMS/Email).
